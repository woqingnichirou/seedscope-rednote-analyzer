# SeedScope Roadmap

SeedScope 的路线图围绕中国大陆品牌营销场景展开，重点是本地优先、截图分析、内容策略标签、品牌安全和可复用报告。

## v0.1 MVP

目标：完成本地可运行的端到端最小产品。

- Next.js 前端基础流程：首页、创建项目、上传、校正、标签、报告。
- FastAPI 后端基础接口：项目、上传、OCR、标签、报告、导出。
- SQLite + SQLModel 本地存储。
- OCR 优先 PaddleOCR，失败时支持 Tesseract fallback。
- Jinja2 生成 Markdown 和 HTML 报告。
- openpyxl 导出 Excel 明细。
- `packages/rules` 第一版规则库。
- `examples` 脱敏 demo 数据和 demo 报告。
- README、LICENSE、基础合规说明。

## v0.2 中文化与品牌营销增强

目标：让中国大陆品牌方、代理商和内容团队更容易理解和使用。

- 中文 README 与中文快速启动文档。
- 中国大陆用户 Windows 本地运行指南。
- 典型使用场景文档。
- 更贴近品牌营销语境的内容类型、标题、封面、正文、CTA 和风险规则。
- Demo 数据导入入口。
- OCR 结果批量编辑体验优化。
- 报告章节支持中文业务术语配置。
- 增加产品截图占位和演示说明。

## v0.3 策略洞察与素材复投

目标：从“生成报告”升级为“沉淀内容资产和复投素材池”。

- LLM 洞察生成，支持严格 JSON schema。
- OpenAI provider 完整配置。
- DeepSeek / Claude provider 接口落地。
- 高赞内容类别自动归因。
- 达人能力标签和高决策作者评分。
- 可复投素材白名单。
- 按达人、选题、封面、CTA 的周度复盘看板。
- 风险表达检查和替代表达建议。
- 更完整的报告历史和多项目对比。

## v1.0 团队协作与生产化

目标：支持团队在真实品牌内容流程中稳定使用。

- 多项目管理和项目归档。
- 团队协作权限。
- 可配置行业模板。
- 可配置报告章节和导出样式。
- 更稳定的 OCR pipeline 和任务队列。
- 本地部署与私有化部署文档。
- 自动化测试覆盖 API、前端流程、报告模板和规则库。
- 品牌安全检查清单。
- 版本化规则库和模板市场。

## 长期原则

- 不做爬虫。
- 不绕过平台限制。
- 不处理未授权隐私数据。
- 本地优先。
- 报告结论可解释、可复核、可导出。
