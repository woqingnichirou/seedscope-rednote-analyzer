Pain Points

核心痛点

01

人工看截图慢，互动数据和内容结构难统一整理。

02

分析不成体系，容易停留在“谁赞多、谁发得多”。

03

报告难复用，每次月报、提案、复盘都要重做。

04

策略难落地，标题、封面、正文和 CTA 没有可执行标签。

Core Features

核心功能

项目管理

创建 Brand A vs Brand B 分析项目，记录行业、周期和分析目标。

截图 OCR

从用户自有截图中提取标题、封面文案、互动数据、发布时间和账号信息。

人工校正

通过可编辑表格修正 OCR 结果，确保报告输入可信。

内容标签

自动判断内容类型、标题套路、封面表达、正文结构、CTA 和风险。

竞品报告

按品牌营销语境生成结论先行的竞品种草分析报告。

多格式导出

支持 Markdown、HTML、Excel、Word 和 PPT，便于内部汇报与二次编辑。

Privacy & Compliance

隐私与合规

No scraping：不做爬虫，不绕过平台限制，不做登录自动化。

Local-first：本地优先运行，上传文件和导出结果默认保存在用户本机。

用户自有截图：只处理用户有权使用的截图素材，不内置真实平台截图。

Brand-safe：示例数据使用 Brand A / Brand B，不包含真实预算、达人或敏感信息。

Quick Start

快速开始

Windows 非技术用户

下载项目后，双击运行安装脚本和启动脚本。

scripts/windows/setup_windows.bat scripts/windows/start_windows.bat

开发者

使用 Docker Compose 或本地前后端命令启动。

docker compose up --build http://127.0.0.1:3000
